# Culinaire

Culinare is a web application dedicated to elevated home cooking.  
The platform brings classic and fine-dining recipes into home kitchens, making sophisticated cuisine accessible to non-professional cooks through clear, structured instructions and elegant presentation.

## Concept

Culinare focuses on:
- classic and high-end recipes with restaurant-style presentation  
- step-by-step instructions designed for home cooks  
- curated content created in collaboration with professional chefs  

Each recipe is presented as both a cooking guide and a culinary story.

## Core Features (MVP)

- Recipe catalog with categories (meat, fish, desserts, soups & basics)
- Detailed recipe pages with:
  - ingredients
  - preparation steps
  - cooking time & difficulty level
- User authentication
- Favorites list for registered users
- Shopping list generated from selected recipes
- Admin panel for managing:
  - categories
  - chefs
  - recipes

## Target Audience

- Home cooks who want to prepare refined, restaurant-style dishes
- Professional chefs and restaurants who want to showcase their recipes and culinary identity
